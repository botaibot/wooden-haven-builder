
# WoodTenerife Project

## Проект строительства и продажи деревянных домов на Тенерифе

Данный проект представляет собой веб-сайт компании WoodTenerife, специализирующейся на проектировании, строительстве и продаже экологичных деревянных домов на острове Тенерифе.

## Технологии

Проект построен с использованием:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Основные разделы

- Главная страница
- Каталог домов
- Строительные материалы
- Калькулятор стоимости
- О компании
- Контакты

## Локальное развертывание

```sh
# Шаг 1: Клонировать репозиторий
git clone <URL_РЕПОЗИТОРИЯ>

# Шаг 2: Перейти в директорию проекта
cd woodtenerife

# Шаг 3: Установить необходимые зависимости
npm i

# Шаг 4: Запустить сервер разработки
npm run dev
```
